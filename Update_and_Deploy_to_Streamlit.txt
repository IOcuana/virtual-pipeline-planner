Virtual Pipeline Planner – Update & Deploy Guide (Streamlit Cloud)
=================================================================

Purpose
-------
This cheat‑sheet shows how to edit your local .py file, test it, push to GitHub,
and let Streamlit Cloud auto‑redeploy your app.

Prereqs (one-time)
------------------
1) Git installed and on PATH (verify: `git --version`).
2) GitHub repository exists: https://github.com/IOcuana/virtual-pipeline-planner
3) Streamlit Cloud app is deployed and points to:
   - Repo: IOcuana/virtual-pipeline-planner
   - Branch: main
   - Main file: cng_planner_app_updated_v13_9.py
4) Optional local venv for testing:
   - Create once: `python -m venv .venv`
   - Activate: `.\.venv\Scripts\activate`
   - Install: `python -m pip install -r requirements.txt`

Quick Update Flow (every time you make changes)
-----------------------------------------------
1) Edit your file locally in PyCharm/VS Code:
   C:\Users\TysonCooneyH2Hauler\Development 1A\Virtual Pipeline\cng_planner_app_updated_v13_9.py

2) (Optional) Test locally before pushing:
   .\.venv\Scripts\python.exe -m streamlit run cng_planner_app_updated_v13_9.py
   - Stop with Ctrl+C when done.

3) Stage, commit, and push to GitHub (from your project folder):
   git status                      (see what changed)
   git add cng_planner_app_updated_v13_9.py
   git commit -m "Update: <brief description>"
   git push

   Tip: To stage everything including deletes/renames, use:
   git add -A

4) Wait for Streamlit Cloud to auto‑redeploy:
   - Open your app URL: https://iocuana-virtual-pipeline-planner.streamlit.app
   - It will show “Updating…” during the rebuild (usually 1–3 minutes).

5) Verify changes in the browser.
   - If something looks off, fix locally and repeat steps 1–4.

One‑liner Push (optional convenience)
-------------------------------------
Save this as PushUpdate.bat in your project folder:

@echo off
set MSG=%*
if "%MSG%"=="" set MSG=Quick update
git add -A
git commit -m "%MSG%"
git push

Run it like:
PushUpdate.bat "v13.10: planner ribbon tweak"

Troubleshooting
---------------
• Streamlit says “Branch/File does not exist”
  - Make sure you pushed to the correct repo/branch.
  - Confirm on GitHub the file path matches: cng_planner_app_updated_v13_9.py

• Permission/Authentication prompts on push
  - Choose “Sign in with browser” when prompted, or use a GitHub Personal Access Token.

• App didn’t update after push
  - Refresh the app page; check Streamlit Cloud logs (hamburger menu → Settings → Logs).
  - Confirm you pushed to `main` (the branch your app uses).

• Changing main file or branch later
  - In Streamlit Cloud → “My apps” → your app → “Edit app” and update the settings.

End of guide.
